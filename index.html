Multi-User, Multi-Business Accounting Application

Quick Start

This application is containerized with Docker. To run it:

  1. Ensure Docker and Docker Compose are installed
  2. Run: docker-compose up --build
  3. Access the application at http://localhost:3000

Services

  • Frontend: http://localhost:3000
  • Backend API: http://localhost:5000
  • MinIO Console: http://localhost:9001

Features

  • Multi-business support with role-based access control
  • Chart of accounts management

  • Inventory management with weighted-average costing
  • Fixed asset tracking with depreciation and capital allowance
  • Comprehensive tax calculations (VAT, WHT, PAYE, CIT, PIT)

  • Financial reports and tax computations
